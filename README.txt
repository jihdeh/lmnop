UberClone
----------------------------

Hello and thank you for purchasing this project, if you have any issues starting up this project, please feel free to reach out to me @ jide.b.tade@gmail.com.

In order to start this project, it is assumed you are familiar with ReactJS and ReactNative, if you aren't please visit the links at the end of this document, as it will guide you to getting started.

Please note that this project doesn't contain a backend nor does it connect to a dynamic map, you would have to setup the rest of the configurations yourself. this just sets up the basic uber like interface.

Getting Started.
...................................

1. Open up the project and then in your terminal `cd` into the folder
2. Run `yarn install` if you already use yarn, or run `npm install`
3. To run on IOS simulator, run `react-native run-ios` or `react-native run-android`
